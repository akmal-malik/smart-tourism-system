# 🌍 Smart Tourism Management System

A full-stack web application designed to improve tourism management by enabling seamless booking, real-time data access, and transparent data request handling.

---

## 🚀 Problem Statement

Tourism offices often delay providing data due to workload, requiring multiple follow-ups. This system eliminates that friction by making tourism data accessible and automating requests.

---

## 💡 Solution

This platform provides:

- 📅 Online booking system for visitors  
- 📊 Real-time tourism data dashboard  
- 📩 Data request system for users  
- 🛠 Admin dashboard for managing bookings and requests  

---

## 🧱 Tech Stack

**Frontend**
- React.js
- Tailwind CSS
- React Router

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB Atlas

---

## ✨ Features

### 👤 User Side
- Book visits to Bangkong Kahoy Valley  
- View tourism data instantly  
- Submit data requests  

### 🛠 Admin Side
- View all bookings  
- Monitor user requests  
- Analyze tourism data  

---

## 📊 Data Dashboard

Displays:
- Total bookings  
- Total visitors  

👉 Built using real-time data from MongoDB

---

## 🧩 System Architecture

Frontend (React) → Backend (Express API) → Database (MongoDB)
